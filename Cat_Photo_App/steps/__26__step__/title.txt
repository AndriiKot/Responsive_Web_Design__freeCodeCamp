Step  26

A figure caption (figcaption) element 
is used to add a caption to 
describe the image contained within the figure 
element.
Here is an example of a figcaption 
element with the caption of A cute 
cat:
Example Code&lt;figure&gt;
  &lt;img src="image.jpg" alt="A description 
of the image"&gt;
  &lt;figcaption&gt;A cute cat&lt;/figcaption&gt;
&lt;/figure&gt;

After 
the image nested in the figure element, 
add a figcaption element with text set 
to:
Cats love lasagna.
